dotfiles
========
## general
- **distro**: gentoo
- **shell**: zsh
- **font**: JetBrains Mono
- **wm**: i3-gaps
- **colorscheme**: catppuccin

## details
- **panel**: polybar
- **launcher**: rofi
- **terminal**: kitty
- **browser**: ungoogled chromium
- **music**: ncmpcpp
- **file manager**: ranger
- **editor**: neovim
- **screenshot tool**: maim
- **pdf**: zathura
- **compositing**: picom
- **wallpapers & image viewer**: feh
